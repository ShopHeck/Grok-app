# Extension Icon Assets

Place the following icon files here before publishing to Chrome Web Store:

- `icon-16.png` (16x16px)
- `icon-32.png` (32x32px)
- `icon-48.png` (48x48px)
- `icon-128.png` (128x128px)

These should be the AgentDesk lightning bolt logo on a transparent or branded background.

## Design Specs

- Format: PNG with transparency
- Color: #6366F1 (primary indigo) on white or transparent
- Shape: Rounded square (for store listing) + raw icon (for toolbar)
